1.การทำงานของ All Server
Server ทุกตัวทำงานปกติหรือไม่
Main Server Active Directory, File Share ทำงานปกติหรือไม่
Check Antivirus Server (KSCSV) ทำงานปกติหรือไม่
Mail Server Check Server Exchange ทำงานปกติหรือไม่
VM แต่ละตัวทำงานปกติหรือไม่
ตรวจเช็ค SAN ว่าทำงานปกติหรือไม่
ตรวจเช็ค Data Domain ว่าทำงานปกติหรือไม่
ตราวจเช็คระบบ CCTV ว่ามีการทำงานตามปกติหรือไม่

2.Fire wall, Backup data
ตรวจเช็ค Log จาก Fire Wall มีสิ่งผิดปกติหรือไม่
ตรวจเช็คการ Backup Data ของ PPDM(Power Protect Data Manager) เป็นปกติหรือไม่

3.การป้องกันอัคคีภัย
มีวัตถุไวไฟหรือวัตถุที่เป็นเชื้อเพลิงในห้อง Server หรือไม่
ปลั๊กไฟเป็นระเบียบ, ปลั๊กหลวม, ปลั๊กชำรุด และมีความปลอดภัยหรือไม่
Check UPS Overload, Energy Down

4.เสียง
ตรวจเช็คว่าเครื่อง Server มีเสียงดังผิดปกติหรือไม่ เช่น Fan, Hard Disk, Alert

5.การรั่วซึมของน้ำ
ตรวจสอบการรั่วซึมของน้ำ เช่น น้ำซึมจากแอร์, ฝ้า-เพดาน, พื้น, ผนัง

6.Network
การตรวจสอบการทำงานของ Switch Hub ว่ามีการเปิดไฟและมีการกระพริบของไฟทุก Port หรือไม่
ตรวจสอบการทำงานของ Media Converter ว่า Access LED มีการเปิดไฟกระพริบหรือไม่

7.การทำความสะอาด
เช็คฝุ่น Server, Switch Hub, Rack
